                                          Hello.

                                          Dto is short for Data Transfer Object and additional T is here because this param is generic one, and usually the "T" letter is used for the generic parameters (for example if you take a look at the implementation of generic List in C# you will see the List<t>). That's the reason why the "T" letter is placed in front of each parameter TEntity and TDto.
